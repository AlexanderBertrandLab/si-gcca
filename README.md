## About

This repository includes the MATLAB-code for the SI-GCCA algorithm as explained in [1] as well as all the experiments from the paper in [1], conducted on the publicly available dataset of [2]. 

Developed and tested in MATLAB R2021b.

Note: Tensorlab is required (https://www.tensorlab.net/).
